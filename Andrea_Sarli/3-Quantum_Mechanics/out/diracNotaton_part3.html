Operator on a system of two Qubits

When applying an operator such as the Pauli-Z operator σz\sigma_zσz​ to a system of two qubits, its effect depends on which qubit it acts upon.

For example:

  • Applying σz\sigma_zσz​ to the first qubit in the tensor product state ∣ψ⟩=∣q1⟩⊗∣q2⟩|\psi\rangle = |q_1\rangle \otimes |q_2\rangle∣ψ⟩=∣q1​⟩⊗∣q2​⟩ is written as:

    (σz⊗I)∣q1⟩⊗∣q2⟩=(σz∣q1⟩)⊗∣q2⟩(\sigma_z \otimes I) |q_1\rangle \otimes |q_2\rangle = (\sigma_z |q_1\rangle) \otimes |q_2\rangle (σz​⊗I)∣q1​⟩⊗∣q2​⟩=(σz​∣q1​⟩)⊗∣q2​⟩

  • Applying σz\sigma_zσz​ to the second qubit is written as:

    (I⊗σz)∣q1⟩⊗∣q2⟩=∣q1⟩⊗(σz∣q2⟩)(I \otimes \sigma_z) |q_1\rangle \otimes |q_2\rangle = |q_1\rangle \otimes (\sigma_z |q_2\rangle) (I⊗σz​)∣q1​⟩⊗∣q2​⟩=∣q1​⟩⊗(σz​∣q2​⟩)

The Pauli-Z operator flips the phase of the

∣1⟩|1\rangle∣1⟩

state:

σz∣0⟩=∣0⟩\sigma_z |0\rangle = |0\rangle σz​∣0⟩=∣0⟩

σz∣1⟩=−∣1⟩\sigma_z |1\rangle = -|1\rangle σz​∣1⟩=−∣1⟩

This means that if σz\sigma_zσz​ acts on a basis state like ∣01⟩|01\rangle∣01⟩, the result is:

∣0⟩⊗(σz∣1⟩)=−∣01⟩|0\rangle \otimes (\sigma_z |1\rangle) = -|01\rangle ∣0⟩⊗(σz​∣1⟩)=−∣01⟩

We can also understand the action of the Pauli-Z operator on multi-qubit basis states. Recall the following:

σz∣↑⟩=+1∣↑⟩\sigma_z |\uparrow\rangle = +1 |\uparrow\rangle σz​∣↑⟩=+1∣↑⟩

σz∣↓⟩=−1∣↓⟩\sigma_z |\downarrow\rangle = -1 |\downarrow\rangle σz​∣↓⟩=−1∣↓⟩

This can also be written in computational basis as:

σz∣0⟩=∣0⟩\sigma_z |0\rangle = |0\rangle σz​∣0⟩=∣0⟩

σz∣1⟩=−∣1⟩\sigma_z |1\rangle = -|1\rangle σz​∣1⟩=−∣1⟩

Application to σz\sigma_zσz​

For the Pauli-Z operator, the eigenstates and eigenvalues are:

σz∣0⟩=+1∣0⟩,σz∣1⟩=−1∣1⟩\sigma_z |0\rangle = +1 |0\rangle,\quad \sigma_z |1\rangle = -1 |1\rangle σz​∣0⟩=+1∣0⟩,σz​∣1⟩=−1∣1⟩

Given the state:

∣ψ⟩=a∣0⟩+b∣1⟩|\psi\rangle = a|0\rangle + b|1\rangle ∣ψ⟩=a∣0⟩+b∣1⟩

We compute the measurement probabilities:

∣⟨0∣ψ⟩∣2=∣a∣2,∣⟨1∣ψ⟩∣2=∣b∣2|\langle 0 | \psi \rangle|^2 = |a|^2,\quad |\langle 1 | \psi \rangle|^2 = |b|^2 ∣⟨0∣ψ⟩∣2=∣a∣2,∣⟨1∣ψ⟩∣2=∣b∣2

⟨σz⟩=(+1)⋅∣a∣2+(−1)⋅∣b∣2=∣a∣2−∣b∣2\langle \sigma_z \rangle = (+1) \cdot |a|^2 + (-1) \cdot |b|^2 = |a|^2 - |b|^2 ⟨σz​⟩=(+1)⋅∣a∣2+(−1)⋅∣b∣2=∣a∣2−∣b∣2

This result is consistent with the one obtained using the direct matrix operation, confirming that expectation values can be understood as weighted averages of the eigenvalues.
